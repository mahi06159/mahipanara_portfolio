Mailbox

The application provides secure user registration with password hashing, session-based login, and the ability to compose, send, and manage messages. The project features a responsive frontend built with Bootstrap and Jinja2, demonstrating a complete client-server architecture.

TechniCart

This E-Commerce Website is a full-stack web application built with the MERN stack (MongoDB, Express.js, React.js, and Node.js). The application allows users to register and log in, browse a dynamic product catalog, manage a shopping cart, and complete a secure checkout process. The project features a decoupled client-server architecture, a responsive React frontend, and stateless user authentication using JSON Web Tokens (JWT).

Global Awareness & Ethical Considerations

I believe that building technology is not just about writing code, but about making responsible choices that shape our future. For me, this means championing ethical principles at every stage of development. I am committed to designing systems with privacy by design, ensuring algorithms are fair and unbiased, and building products that are accessible to everyone, regardless of ability. I also consider the sustainability of technology, aiming to write efficient code that respects our planet's resources.

Furthermore, I recognize that the most impactful solutions must be globally aware and locally relevant. Growing up in a diverse country like India has taught me the importance of considering cultural nuances, economic constraints like low bandwidth, and varying regulatory landscapes. My goal is to build technology that is not only powerful but also equitable, secure, and beneficial for a global society.
